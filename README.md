# Phoosball

Phoosball score tracking webapp written in Go

## set up
1. install go
2. start mariadb docker container (`cd db/ && ./run.sh`)
3. install dependencies (`go get -u ./..`)
4. run the server in debug (test) mode (`go run phoos_server -debug`)
